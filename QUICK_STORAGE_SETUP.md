# ⚡ Quick Storage Bucket Setup (2 minutes)

## Fastest Method: Public Bucket

1. **Go to Supabase Dashboard** → Your Project → **Storage**

2. **Click "New bucket"**

3. **Fill in:**
   - Name: `competition-submissions`
   - ✅ Check **"Public bucket"**
   - File size limit: `10 MB` (optional)

4. **Click "Create bucket"**

5. **Done! ✅**

---

## That's It!

The bucket is now ready to accept competition submissions. Files will be publicly accessible via URL, which is fine for competition submissions.
