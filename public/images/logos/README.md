# Texas A&M Mays Business School Logos

Official branding logos used throughout the CMIS Events website.

## Logo Files

| File | Description |
|------|-------------|
| `atm-mays.jpg` | Texas A&M ATM logo with "MAYS" text |
| `tamu-seal.png` | Texas A&M University official seal (Est. 1876) |
| `cmis-logo.jpeg` | CMIS with ATM branding |
| `mays-horizontal.png` | Horizontal Mays Business School wordmark |

## Usage

These logos appear in:
- **Header** - ATM Mays logo in navigation
- **Footer** - ATM Mays, TAMU seal, and Mays horizontal logos
- **Login & Signup** - TAMU seal and CMIS logo
- **Homepage Hero** - TAMU seal and CMIS logo prominently displayed
